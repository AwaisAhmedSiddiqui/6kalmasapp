# 6 Kalmas App

## Overview
The **6 Kalmas App** is a mobile/web application designed to help users learn and recite the six important Kalmas of Islam. It provides a simple, interactive, and user-friendly interface for learning, memorizing, and listening to the Kalmas.

## Features
- Display all six Kalmas with Arabic text.
- Translation and transliteration for easy understanding.
- Audio playback for proper pronunciation.
- User-friendly and intuitive interface.
- Offline support for learning anytime, anywhere.

## Kalmas Included
1. Kalma Tayyaba
2. Kalma Shahada
3. Kalma Tamjeed
4. Kalma Tauheed
5. Kalma Astaghfar
6. Kalma Radde Kufr
